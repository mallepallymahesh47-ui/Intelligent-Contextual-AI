# Intelligent Contextual AI (RAG)

Intelligent Contextual AI is a Retrieval-Augmented Generation (RAG) based assistant that combines Google Gemini with Qdrant vector search to enable intelligent document querying and conversational AI.
It supports multiple file formats including PDF, DOCX, XLSX, and TXT, allowing users to chat with their documents or switch to normal LLM mode. Providing real-time tracing, debugging, and monitoring of LLM interactions.

## Features

- Chat with Gemini LLM (Normal Mode)
- Document-based Q&A using RAG
- Supports PDF, TXT, DOCX, XLSX files
- Qdrant Cloud vector database integration
- Clean Streamlit UI with mode toggle
- LangSmith observability, Tracing & Monitoring

## Tech Stack

- Python
- Streamlit
- Qdrant (Vector Database)
- Google Gemini (LLM)
- Sentence Transformers (Embeddings)
- Langsmith
- Langchain
